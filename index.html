Artikel: Tatacara Pembuatan Prototype Smart Home

1️⃣ Siapkan Komponen ESP32 Relay module Lampu (lebih aman pakai lampu DC kecil untuk prototype) Kabel jumper

2️⃣ Rangkai ESP32 3.3V/5V → VCC relay ESP32 GND → GND relay ESP32 GPIO 23 → IN relay Kabel lampu masuk lewat relay (COM → NO). ⚠️ Jika lampu 220V, minta bantuan orang dewasa.

3️⃣ Upload Program Gunakan kode paling sederhana ini: #include #include WebServer server(80); const int relayPin = 23; void setup() { pinMode(relayPin, OUTPUT); digitalWrite(relayPin, HIGH); WiFi.softAP("ESP32-AP", "12345678"); server.on("/on", [](){ digitalWrite(relayPin, LOW); server.send(200, "text/plain", "ON"); }); server.on("/off", [](){ digitalWrite(relayPin, HIGH); server.send(200, "text/plain", "OFF"); }); server.begin(); } void loop() { server.handleClient(); }

4️⃣ Kontrol dari HP Sambungkan HP ke Wi-Fi ESP32-AP Buka browser → ketik 192.168.4.1/on (lampu ON) Ketik 192.168.4.1/off (lampu OFF)
